# Portugal Compliance App

This app provides features for Portuguese fiscal compliance within ERPNext, including:

*   SAF-T (PT) file generation
*   ATCUD generation and validation
*   QR Code generation for invoices
*   Document signing (placeholder)
*   Compliance audit logging

## Installation

Follow the standard Frappe app installation procedure.

## Usage

Refer to the User Guide and Technical Guide in the `docs/` folder.
